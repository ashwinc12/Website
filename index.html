Poll Buddy
January 2020 - Present

Through an open-source class/club at RPI, Rensselaer Center for Open Source (RCOS), I joined the Poll Buddy project. Poll Buddy is an affordable and compatible in-class quizzing platform that is meant to replace iClicker usage at RPI. The frontend was created using React along with MDB containers, while the backend uses Node.js with a MongoDB database. My main contribution to the project was designing and developing multiple frontend pages. Poll Buddy is tentavively scheduled for release for use Fall 2020. More about the project can be found here.






CourseHub
November 2019

As a project for HackRPI 2019, my group members and I developed CourseHub, an automatic lecture recording system. The goal of CourseHub is to allow students to easily access lectures recordings after class in order to review material, catch up, etc. This was built using a Flask frotnend framework, a Raspberry Pi, and a Google Cloud database. The Raspberry Pi is able to automatically record a webcam using OpenCV, and upload the video to Google Cloud, where it can be instantly accessed by students. More info about the project can be found here.
